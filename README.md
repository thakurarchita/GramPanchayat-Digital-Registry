# Gram Panchayat Digital Registry

An Android application developed to digitize Gram Panchayat administrative services and citizen records. The application streamlines record management, improves accessibility, and integrates an AI-powered chatbot to assist users with common queries.

---

## Features

- Secure Admin and User Authentication
- Citizen Registration
- Birth & Death Record Management
- Document Management
- AI Chatbot Assistance
- User-Friendly Android Interface
- Database Integration
